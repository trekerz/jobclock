# Treker's 就业小闹钟 (jobclock)

http://www.jobclock.xyz



### 这是一个为华南理工大学13级同学在2016年求职季准备的一个宣讲会信息发布网站。



这是一个低成本小项目网站：

​	**服务器：**腾讯云（月供）；

​	**域名：**万网（暂购1年）；

​	**架构：**LAMP；

​	**系统：**CentOS 6.5；

​	**配置：**20G + 1G + 单核；



此网站数据来源于华南理工大学就业信息网（ http://www.scut.edu.cn/bys ），网页默认展示以当天为中心的10天宣讲会信息，本人作为管理员会保持信息日更。
